# 06. 无认证模式（内网部署）

> 来源：用户决策（"这个工具在内网运行，不考虑做登录页"）

## 6.1 设计原则

- **免登录**：本系统作为个人局域网/私有云服务运行，不需要任何密码验证或登录表单。
- **无用户数据**：删除 `User` 表，所有存储在数据库中的偏好和历史均隐含关联到唯一的隐式用户。
- **直连可用**：首次启动或重启后，用户通过浏览器访问，直接看到视频库首页。

## 6.2 路由权限调整

- 后端的所有 `/api/` 路由均为**公开路由（Public）**，不加任何 JWT / Cookie 验证。
- 绑定地址为 `0.0.0.0`，任何人只要能通过 IP/局域网域名访问，即可直接控制下载和观影。

## 6.3 局域网安全提示（文档记录）

若后续需要通过公网安全访问该内网工具，推荐在外部加设代理保护：

1. **Basic Auth**：在 Nginx / Caddy 代理层添加基础认证密码保护。
2. **Tailscale / VPN**：只允许在虚拟局域网内访问。
3. **Cloudflare Tunnel**：配合 CF Access 单点登录。
